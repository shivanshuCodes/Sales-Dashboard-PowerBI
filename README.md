Project Overview This dashboard analyzes sales data from 2020–2022 using 3 structured Excel tables. The report dynamically updates based on slicer selections, allowing users to explore trends, performance, and key metrics interactively.

Tools & Technologies Used

MS Excel – Data Cleaning & Preprocessing
Power BI – Data Modeling & Visualization
DAX – KPI & Measure Creation
Dataset Information

3 Dummy Sales Data Tables (Excel)
Cleaned and formatted before importing into Power BI
Data includes: Sales Cost Profit Quantity Transactions Category Brand State
Key Features

✔ Interactive & Responsive Dashboard ✔ Dynamic KPI Cards (Sales, Profit, Quantity, Transactions) ✔ Category-wise Performance Analysis ✔ Brand-wise Contribution ✔ State-level Sales Visualization ✔ Slicers for filtering data dynamically

Compared yearly sales trends
Evaluated regional sales performance
